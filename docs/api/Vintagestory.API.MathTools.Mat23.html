2x3 Matrix

  • A mat2d contains six elements defined as:
    • 

  • [a, b,
    • 

  • c, d,
    • 

  • tx,ty]
    • 

  • 

    • 

  • This is a short form for the 3x3 matrix:
    • 

    • 

  • [a, b, 0
    • 

  • c, d, 0
    • 

  • tx,ty,1]
    • 

  • 

    • 

  • The last column is ignored so the array is shorter and operations are faster. Open in GitHub
    •
